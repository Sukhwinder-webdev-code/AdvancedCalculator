  ## overview
  The Advanced Calculator Application is a responsive web-based calculator built using HTML, CSS, and JavaScript. It provides users with a fast and intuitive interface for performing mathematical calculations, including addition, subtraction, multiplication, division, percentage operations, and decimal calculations.
The application is designed to simulate the functionality of a standard calculator while offering a clean user experience and accurate calculation handling. This project was developed to strengthen JavaScript programming skills, event handling, DOM manipulation, and logical problem-solving.

## Features
- Addition, subtraction, multiplication, and division
- Percentage calculations
- Decimal number support
- Clear and reset functionality
- Real-time calculation display
- Responsive and user-friendly interface
- Accurate mathematical operations

## tech-stack
- HTML5 – Structure and layout
- CSS3 – Styling and responsive design
- JavaScript (ES6) – Calculator logic and functionality
- Git & GitHub – Version control
- Vercel – Deployment and hosting

## usage
- Enter numbers using the calculator interface.
- Perform addition, subtraction, multiplication, and division operations.
- Calculate percentages and decimal values.
- View results instantly on the display screen.
- Clear calculations and start new operations as needed.
